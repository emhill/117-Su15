Mac UNIX Python Development Setup

Download & install a UNIX-based command line interface and a text editor for python development.

  1. Use spotlight to search your mac for the “Terminal” application.
    • if it’s not there, you will need to install the Xcode dev tools: http://itunes.apple.com/us/app/xcode/id497799835?ls=1&mt=12
  2. Get the latest version of python 3 https://www.python.org/downloads/
  3. TextWrangler editor: http://www.barebones.com/products/textwrangler/download.html
