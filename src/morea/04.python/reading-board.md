## Python arithmetic 2/6

Python supports 6 basic arithmetic operations:

 * addition (+)
 * subtraction (-)
 * multiplication (*)
 * floating point division (/)
 * integer division (//) -- rounds down to nearest whole number
 * exponent (**)
 * remainder or modulus (%) -- returns remainder after division
